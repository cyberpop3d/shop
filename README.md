# CyberPop Shop v17 — Fight Game UI Reset

A clean Next.js prototype for the CyberPop Shop home interface.

## What changed

- Full fight-game-inspired UI reset
- One selected collectible focus instead of a two-character versus layout
- Square cover images everywhere: roster, collection, loot, rewards, unlocks
- Cover image is the default preview; 3D model only appears after the user clicks **3D print preview**
- Roster uses a fighter-select style scrollable portrait grid
- Rewards page is rebuilt around a compact monthly poster plus progress track checkpoints
- Purchase panel keeps GET STL and GET Physical Product flows
- Loot section uses a single center crystal and connected character ring

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
