# CyberPop Shop Game UI Prototype

Clean Next.js prototype for the CyberPop Shop interface.

## Run locally

```bash
npm install
npm run dev
```

## Deploy

Upload the contents of this folder to the GitHub repository, then connect it to Vercel.

This build intentionally avoids Tailwind, Framer Motion, Lucide, and strict TypeScript to reduce deploy friction.


## v8 note

Sonya Blade includes a live GLB preview at `public/models/sonya-blade.glb`. The app loads Google's `<model-viewer>` web component from CDN for this prototype.
